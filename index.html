Custom Buttons (Implicit)

Note: Does not work in IE 6 - Please join the movement.

Implicitly styled button elements and class variations.

<button><span>Button</span></button>
<button type="submit"><span>Submit</span></button>
<button disabled="disabled"><span>Disabled</span></button>
<button class="primary"><span>Primary</span></button>

Icon or image button.

<button><span><img src="img/icon.png" alt="" />Icon button</span></button>
<button type="submit"><span><img src="img/icon.png" alt="" />Submit</span></button>
<button disabled="disabled"><span><img src="img/icon.png" alt="" />Disabled</span></button>
<button class="primary"><span><img src="img/icon.png" alt="" />Primary</span></button>
